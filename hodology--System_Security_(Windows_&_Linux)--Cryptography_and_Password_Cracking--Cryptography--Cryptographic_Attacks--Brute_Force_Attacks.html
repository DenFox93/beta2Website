Brute Force Attacks


A brute force attack generate and test every combination of the key

Most is an attack of known plaintext only attack or known ciphertext only attack when the attacker can easily verify the correctness of the guess.

weakness: time to find the password
pseudo-code of a brute force attack:
password_found = false
password_length = 1
while (password_found == false) do
    while (can_create_password_of_length(password_length)) do
        password = create_password_of_length(password_length)
        if (hash(password) match attacked_hash)
            password_found = true
    done
    password_length = password_length + 1
done


A brute force attack is worth using when we are in a penetration testing engagement and:
• we don't have anything to crack
• the wordlist failed
