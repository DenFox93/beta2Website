hrSWInstalledName OID



Software installed on the machine



snmpwalk -c public -v2c 192.168.1.43 hrSWInstalledName

-v → specifies the SNMP version to use ( 1 )
-c → sets the community string to use ( public ), to discover which is it we can use the snmp-win32-services NSE script
hrSWInstalledName → specify this OID to list only the software installed on the machine.
numerically this is the value 1.3.6.1.2.1.25.6.3.1.2
