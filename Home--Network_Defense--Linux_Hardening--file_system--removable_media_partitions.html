removable media partitions



Removable Media also have Partitions and usually (except Floppy Disk) they use the same structures that hard disk use.

Removable media containing character and block special devices could be used to circumvent security controls by allowing non-root users to access sensitive device files such as /dev/kmem or the raw disk partitions.

• nodev option set on
nodev mount option specifies that the filesystem cannot contain special devices.
You may still be able to create device nodes, but if the filesystem is mounted with the nodev option, any device nodes on that filesystem won't work.(stackexchange)

• noexec option set on
