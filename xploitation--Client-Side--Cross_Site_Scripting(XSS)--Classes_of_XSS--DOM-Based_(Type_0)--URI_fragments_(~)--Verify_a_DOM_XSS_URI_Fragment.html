Verify a DOM XSS URI Fragment



Below points are true only for DOM XSS URI fragment vulnerability and NOT for DOM XSS in general

DOM XSS URI fragment is a form of XSS where the entire tainted data flow from source to sink takes place in the browser, i.e., the source of the data is in the DOM, the sink is also in the DOM, and the data flow never leaves the browser!
