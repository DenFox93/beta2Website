2. Provide Potentially Vulnerable Endpoints with Internal URLs



Depending on the network configuration, you might need to try several addresses before you find the ones in use by the network.
Common Ip Addresses reserved for the private network:
◇ localhost
◇ 127.0.0.1
◇ 0.0.0.0/8 → current network
◇ 192.168.0.0/16 → private network
◇ 198.18.0.0/15 → private network
◇ 10.0.0.0/8 → private network
◇ 100.64.0.0/10 → private network
◇ For more: https://en.wikipedia.org/wiki/Reserved_IP_addresses


Example of Tests
• webhooks
POST /webhook
Host: public.example.com

url=https://192.168.0.1

• file uploads
POST /upload_profile_from_url
Host: public.example.com

user_id=1234&url=https://192.168.0.1

• document and image processors
• link expansions or thumbnails
• proxy services
https://public.example.com/proxy?url=192.168.0.1

• user-provided URLs
