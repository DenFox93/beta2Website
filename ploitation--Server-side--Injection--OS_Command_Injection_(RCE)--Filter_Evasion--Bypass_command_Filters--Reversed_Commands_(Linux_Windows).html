Reversed Commands (Linux Windows)




Linux
1. To create the reversed command
echo 'whoami' | rev


2. Execute the reverse command in a subshell $()
$(rev<<<'imaohw')




Windows
1. To create the reversed command
"whoami"[-1..-20] -join ''


2. Execute the reverse command in a subshell $()
iex "$('imaohw'[-1..-20] -join '')"
