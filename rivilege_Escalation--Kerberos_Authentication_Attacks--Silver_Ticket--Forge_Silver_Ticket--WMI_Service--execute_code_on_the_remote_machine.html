Execute Code on the Target machine using Invoke-WmiMethod
0. hostname of the Domain Controller(remote machine). In test phase directly on its shell

1. To call WMI with Powershell we can:
Invoke-WmiMethod win32_process -ComputerName <hostname> -name create -argumentlist “<commandToRun>”
2. If inside <commandToRun> we want execute cmd command(inside Powershell) we need to use “cmd.exe /c <cmdCommand>”

example copy a file from a folder to another
PS> Invoke-WmiMethod win32_process -ComputerName testdc -name create -argumentlist “cmd.exe /c copy C:\Users\Administrator\Desktop\Test1\myfile.txt C:\Users\Administrator\Desktop\Test2\myfile.txt”
