IP Address Concealing



Use a proxy to conceal the IP address
In case we want to conceal our IP address, or if a certain web application has a protection mechanism that blacklists our current IP address, we can use
• Proxy
• Tor Network

- A proxy can be set with the option --proxy (e.g. --proxy="socks4://177.39.187.70:33283"), where we should add a working proxy.
In addition to that, if we have a list of proxies, we can provide them to SQLMap with the option --proxy-file. This way, SQLMap will go sequentially through the list, and in case of any problems (e.g., blacklisting of IP address), it will just skip from current to the next from the list.
- Tor network use to provide an easy to use anonymization, where our IP can appear anywhere from a large list of Tor exit nodes.
→ When properly installed on the local machine, there should be a SOCKS4 proxy service at the local port 9050 or 9150.
→ By using switch --tor, SQLMap will automatically try to find the local port and use it appropriately.
→ If we wanted to be sure that Tor is properly being used, to prevent unwanted behavior, we could use the switch --check-tor. In such cases, SQLMap will connect to the https://check.torproject.org/ and check the response for the intended result (i.e., Congratulations appears inside).
