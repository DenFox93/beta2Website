reverse_resolve (meglio dig??)


Description
Conducts a reverse lookup for each IP address to resolve the hostname. Updates the 'hosts' table with the results.

We first need to find the nameserver for our target website
dig elsfoo.com NS



to get the IP from the nameserver
dig -t A ns2.elsfoo.com


to see the modules availables that contain that name
marketplace search reverse_resolve


module to conducts a reverse lookup for each IP address to resolve the hostname. Updates the 'hosts' table with the results.
modules load recon/hosts-hosts/reverse_resolve
