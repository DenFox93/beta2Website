1. Hunt for parameters that fetch external resources


SSRFs occur in features that require visiting and fetching external resources:
• webhooks
POST /webhook
Host: public.example.com

url=https://www.attacker.com

• file uploads
POST /upload_profile_from_url
Host: public.example.com

user_id=1234&url=https://www.attacker.com/profile.jpeg

• document and image processors
• link expansions or thumbnails
• proxy services
https://public.example.com/proxy?url=https://google.com

• user-provided URLs
• URLs embedded in files that are processed by the application
◇ XML files
◇ PDF files
• hidden API endpoints that accept URLs as input
• input that gets inserted into HTML tags
