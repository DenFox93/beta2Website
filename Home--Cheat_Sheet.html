Cheat Sheet



Nmap Initial Enumeration

• quick initial nmap scan
nmap -sC -sV -O -oA initial 10.10.10.17


• Scan basic scripts open ports
nmap -sC -sV -p 21,22,55 -oA openPorts 10.10.10.17

• scan for all tcp ports
nmap -p- -oA tcp 10.10.10.17

• scan for all udp ports 
nmap -sU -p- -oA udp 10.10.10.17


• Scan basic scripts 
nmap -sC -sV -p- -oA full 10.10.10.17


Nmap script open ports (default service)
ls /usr/share/nmap/scripts/*service*


nmap --script ftp-vsftpd-backdoor -p 21 10.10.10.3


nmap vulnerablities
