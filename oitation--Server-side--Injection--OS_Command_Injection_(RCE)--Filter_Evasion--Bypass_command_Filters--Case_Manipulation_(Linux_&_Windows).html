Case Manipulation (Linux & Windows)




Inverting the character cases of a command (e.g. WHOAMI) or alternating between cases (e.g. WhOaMi)

Windows Server
It is easy case, in fact since PowerShell and CMD are case-insensitive we can change the casing of the characters of the command and send it:
WhOaMi
WHOAMI




Linux Server
Linux and a bash shell, are case-sensitive, because of that we have to get a bit creative and find a command that turns the command into an all-lowercase word
Note: Because in the commands that we are going to use contains spaces, they can get filtered. Because of that remember the chapter Bypass Space Filters and the possible bypass for the space like %0a %09

• tr command to convert uppercase to lowercase
$(tr "[A-Z]" "[a-z]"<<<"WhOaMi")


• printf command to convert uppercase to lowercase
$(a="WhOaMi";printf %s "${a,,}")
