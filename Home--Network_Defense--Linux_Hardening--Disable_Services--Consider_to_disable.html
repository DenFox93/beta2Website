Consider to disable:




If a service is not necessary it is recommended that the service should be removed to reduce the remote attack surface.
To remove a service:
apt purge [service]


For specific recommendations download the appropriate CIS benchmark of the OS.

You need to consider to disable:
• NFS & SAMBA file sharing are dangerous for servers connected to the internet
To check if they are not installed
dpkg -s samba | grep -E '(Status:|not installed)'
dpkg -s nfs-kernel-server | grep -E '(Status:|not installed)'



• RPC services (example Portmapper)
unprotected rpc services could be dangerous
Verify rpcbind is not installed
dpkg -s rpcbind | grep -E '(Status:|not installed)'

Verify portmap is not installed
dpkg -s portmap | grep -E '(Status:|not installed)'


• Disable HTTP Server (if it is not a Web server)
Verify HTTP servers are not installed
dpkg -s apache2 | grep -E '(Status:|not installed)'

dpkg -s nginx | grep -E '(Status:|not installed)'


• Disable Network Printers (if the host does not need to print)
The Common Unix Print System (CUPS) provides the ability to print to both local and network printers.
A system running CUPS can also accept print jobs from remote systems and print them to local printers. It also provides a web based remote administration capability.
dpkg -s cups | grep -E '(Status:|not installed)'


• SNMP
Simple Network Management Protocol (SNMP) is a widely used protocol for monitoring the health and welfare of network equipment, computer equipment and devices like UPSs.
The SNMP server can communicate using SNMPv1 , which transmits data in the clear and does not require authentication to execute commands. SNMPv3 replaces the simple/clear text password sharing used in SNMPv2 with more securely encoded parameters.
If the the SNMP service is not required, the net-snmp package should be removed to reduce the attack surface of the system.
Note: If SNMP is required:
◇ The server should be configured for SNMP v3 only. User Authentication and Message Encryption should be configured.
◇ If SNMP v2 is absolutely necessary, modify the community strings' values.
Run the following command to verify snmpd is not installed:
dpkg -s snmpd | grep -E '(Status:|not installed)'


• Power/Volume managers
If convenience features like power manager and volume manager are not necessary.
These managers can allow users with phisical access to compromise the system with tools and set-UID programs brought it with it on USB and DVD

• Remove GUI
If a Graphical User Interface (GUI) is not required, it should be removed to reduce the attack surface of the system.
Check if gdm3 is not installed
dpkg -s gdm3 | grep -E '(Status:|not installed)'


• Remove unused login channels
