/tmp


The /tmp directory is a world-writable directory used for temporary storage by all users and some applications
Making /tmp its own file system allows an administrator to set the noexec option on the mount, making /tmp useless for an attacker to install executable code.

• resize
resize the space available for /tmp so that there is no need of resource exaustion
• nodev option set on
nodev mount option specifies that the filesystem cannot contain special devices.
You may still be able to create device nodes, but if the filesystem is mounted with the nodev option, any device nodes on that filesystem won't work.(stackexchange)
• nosuid option set on
• noexec option set on
